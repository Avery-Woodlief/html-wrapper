# html-wrapper
wrappers for tags and document to write html in and upload to a json file for further use

## Tags supported

- html
- meta
- link
- script
- a
- img
- form
- input
- button
- label
- textarea
- select
- option
- iframe
- audio
- video
- div
- header
- nav
- main
- section
- article
- footer
- p
- span
- h1
- h2
- h3
- h4
- h5
- h6

Please refer to lookup.json for more details.
